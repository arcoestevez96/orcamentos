📧 Notificação via Email (Gmail)
Como configurar em 1 minuto:

1. Acesse myaccount.google.com/apppasswords
2. Em "Nome do app" escreva Orçamentos e clique em Criar
3. Copie a senha de 16 letras gerada e cole abaixo
✈️ Notificação via Telegram
Receba uma mensagem no Telegram toda vez que um cliente abrir seu orçamento ou PDF.

Token do Bot: obtido ao criar o bot no @BotFather
Seu Chat ID: o número único da sua conversa com o bot
💬 Notificação via WhatsApp (opcional)
Para receber notificações no WhatsApp quando um cliente abrir seu orçamento, siga os passos:

1. Adicione o número +34 644 60 16 13 na sua agenda do WhatsApp
2. Envie a mensagem: I allow callmebot to send me messages
3. Você receberá seu API Key em alguns segundos
4. Preencha os campos abaixo com seu número e o API Key recebido
🔗 URL Base dos Links
Esta é a URL que aparecerá nos links enviados para seus clientes.

• Uso local: deixe como http://localhost:5000 e use um túnel (veja abaixo)
• Hospedagem: coloque a URL do seu servidor (ex: https://meusite.com)
Como compartilhar links com clientes (uso local):

Abra o Terminal e rode o comando abaixo para criar um link público temporário:

ssh -R 80:localhost:5000 nokey@localhost.run

Cole a URL gerada no campo "URL Base" acima e salve.
